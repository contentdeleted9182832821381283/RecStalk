Getting Set Up
--------------------------------------------------------------------
Go into the Config.Json and change these variables
--------------------------------------------------------------------
Config.Json webhook_url (discord integration webhook)
--------------------------------------------------------------------
Config.Json bearer_token (go to recnet inspect and grab a bearer, log into a acc view a players profile inspect element -> click subscribe (with inspect open) click request and copy the bearer, copy after Bearer and put it into Bearer_Token
--------------------------------------------------------------------
Config.Json Player_Id: Input The Acc Id You would like to monitor grab get accountid from here 
->  https://apim.rec.net/accounts/account?username=USERNAMEHERE
--------------------------------------------------------------------
save it and run it !
